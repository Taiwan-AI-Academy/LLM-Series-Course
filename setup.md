## 如何取得 OpenAI API Key？

1. 前往 [OpenAI Platform](https://platform.openai.com/)
2. 註冊或登入帳號
3. 進入 **API Keys** 頁面
4. 點擊 **Create new secret key**
5. 複製並妥善保存 API Key

> ⚠️ API Key 只會顯示一次，請務必立即保存。
